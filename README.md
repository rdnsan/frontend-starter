## Installation

Install dependencies

```sh
$ yarn
# ... or ...
$ npm install
```

## Run Project Locally

Start the development server

```sh
$ yarn dev
# ... or ...
$ npm run dev

```

Server running at http://localhost:3000

## Build

Build for Production

```sh
$ yarn build
# ... or ...
$ npm run build

```
